# api_method - Experimental Code Sandbox

Thư mục này dành cho **thử nghiệm các hướng tiếp cận mới** và **code thử nghiệm** mà chưa được tích hợp vào phiên bản chính.

## Mục đích

- 📝 Viết và kiểm tra các ý tưởng mới
- 🔧 Thử nghiệm các phương pháp khác nhau
- 🧪 Chạy các bài test khác
- 📚 Lưu trữ các script prototype

## Cấu trúc

```
api_method/
├── README.md                    # File này
├── [các file thử nghiệm]
└── [các script prototype]
```

## Lưu ý

- **Không ảnh hưởng đến version chính**: Code ở đây hoàn toàn độc lập
- **Dễ xóa**: Nếu hết hữu ích, chỉ cần delete folder `api_method` - không ảnh hưởng version đã đóng gói
- **Tự do thử nghiệm**: Không cần lo lắng về breaking changes
